# NewGram-API
(Note: This guide is intended for a Russian-speaking audience only)

NewGram is a messenger built using Pocket Code within the Catrobat development environment. I have released a project that allows you to create your own messenger and create forks of it within Catrobat. You can watch the full guide in this video (available only in Russian): https://youtu.be/SnxkUgY0lS0?si=_z4TYdycRkPaSGvO. The guide also uses our NewGram API site for hosting the messenger: https://s1184852.ha006.t.mydomain.zone/pocketcode/. The files include a ready-to-use Catrobat project containing the messenger featured in the guide
