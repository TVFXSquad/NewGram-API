# NewGram-API
(Note: This guide is intended for a Russian-speaking audience only)

NewGram is a messenger built using Pocket Code within the Catrobat development environment. I have released a project that allows you to create your own messenger and create forks of it within Catrobat.

This guide is intended for Russian speakers; here is the video tutorial: https://youtu.be/SnxkUgY0lS0?si=_z4TYdycRkPaSGvO. The hosting and management service website is: https://s1184852.ha006.t.mydomain.zone/pocketcode/. The completed project from the guide is available in the repository.
